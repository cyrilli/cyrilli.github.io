This is the project I did for my master thesis in Mechatronics Engineering at Harbin Institute of Technology, which I think is pretty fun to watch. 

Basically, a neural network is trained to score different grasp poses generated by a heuristic method. A pipeline to collect data and train this neural network is built using [ROS](https://www.ros.org/about-ros/). The robot will try to pick the randomly generated items in a [Gazebo](http://gazebosim.org/) simulation environment, using the grasp poses produced by the heuristic method. Then by judging whether the robot has successfully picked up the item and dropped it in the bin, the grasp pose is labeled as good or bad. The neural network will then be trained using the collected data.
